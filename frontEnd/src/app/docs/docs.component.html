Documentación de la API: The Office

Esta API proporciona información sobre los personajes de la serie The Office. A continuación, se describen las distintas peticiones que puedes realizar al servidor, los campos que contiene la respuesta y ejemplos prácticos.

Obtener todos los personajes

Método: GET

URL: /api/characters

Descripción: Devuelve una lista con todos los personajes.

Respuesta: Un array de objetos, donde cada uno representa un personaje:

Campos:

  • actor → String → Nombre del actor
  • character → String → Nombre del personaje
  • firstAppearance → String → Primera aparición del personaje
  • quotes → Array → Lista de frases mas famosas
  • image → String → URL de la imagen del personaje

Obtener un personaje por nombre

Método: GET

URL: /api/characters/:name

Parámetros: name → String → Nombre del personaje (en minúsculas y sin espacios)

Descripción: Devuelve un objeto con los datos del personaje solicitado.
