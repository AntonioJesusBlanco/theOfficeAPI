Presentación

Hola, me llamo Antonio Jesus Blanco Guillen y soy desarrollador web con enfoque en tecnologías full-stack. Este proyecto es una demostración práctica de mis habilidades utilizando Angular en el frontend y Node.js con Express en el backend.

¿Qué es esta API?

Esta API está dedicada a los personajes de la serie The Office. Proporciona información detallada sobre cada personaje, incluyendo:

  • Nombre del actor y del personaje
  • Primera aparición
  • Frases características y los capítulos en los que aparecen
  • Imagen del personaje

Los datos están en un archivo JSON y se sirven desde un backend en Node.js.

Tecnologías utilizadas

  • Frontend: Angular, Bootstrap
  • Backend: Node.js, Express
  • Otros: TypeScript, JSON como fuente de datos, CORS

Objetivos del proyecto

  • Aprender y practicar el flujo completo de una aplicación API REST + SPA
  • Mostrar habilidades en diseño responsivo y consumo de APIs en Angular
  • Aplicar buenas prácticas de estructura modular
